\chapter{Explanation of paralist-compat.tex}
\label{annex:paralist-compat}

The file \texttt{NOVAthesisFiles/StyFiles/paralist-compat.tex} provides a set of custom list environments that emulate the behavior of the popular \texttt{paralist} package using the more modern and flexible \texttt{enumitem} package.

\section{Emulated Environments}
This file defines four main environments to ensure compatibility with documents or styles that expect \texttt{paralist} features:

\begin{itemize}
    \item \textbf{asparaenum}: Emulates a paragraph-like numbered list. It uses \texttt{enumitem} to set a wide layout with no extra spacing between items (\texttt{itemsep=0pt}) and matches the paragraph skip.
    \item \textbf{inparaenum}: Emulates an in-paragraph (inline) numbered list. It maps directly to \texttt{enumitem}'s \texttt{enumerate*} environment.
    \item \textbf{compactitem}: A version of the standard itemized list with all vertical spacing removed (\texttt{noitemsep}, \texttt{topsep=0pt}, etc.), making it much denser than a standard \texttt{itemize}.
    \item \textbf{compactenum}: A version of the standard numbered list with all vertical spacing removed, similar to \texttt{compactitem} but for \texttt{enumerate}.
\end{itemize}

\section{Purpose}
The primary reason for this file is to allow the \emph{NOVAthesis} template to provide these ``compact'' and ``inline'' list styles without requiring the actual \texttt{paralist} package, which can sometimes conflict with other list-handling packages. It ensures a consistent look and feel for lists throughout the document while using the robust \texttt{enumitem} engine under the hood.

\section{Summary}
In summary, \texttt{paralist-compat.tex} acts as a lightweight shim, providing familiar list environments for users while maintaining the technical advantages of using \texttt{enumitem}.
