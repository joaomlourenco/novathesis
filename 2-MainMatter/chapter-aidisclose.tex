\chapter{Artificial Intelligence Disclosure Statement}
\label{chap:aidisclose}

As Generative Artificial Intelligence (\gls{AI}) becomes increasingly integrated into academic research, transparency regarding its use is essential. The \novathesis{} template provides built-in support for rendering a formal \emph{Artificial Intelligence Disclosure Statement}.

\section{Why Disclose \glsfmtshort{AI} Use?}

Academic integrity requires that authors are transparent about the tools and methods used to produce their work. Disclosing \gls{AI} use:
\begin{itemize}
  \item \textbf{Promotes Transparency:} It allows readers and examiners to understand which parts of the research process were assisted by \gls{AI}.
  \item \textbf{Ensures Accountability:} Regardless of \gls{AI} assistance, the \textbf{author remains solely responsible} for the accuracy, originality, and ethical integrity of the thesis. \gls{AI}-generated “hallucinations” or errors are the author's responsibility to correct.
  \item \textbf{Complies with Regulations:} Many universities and publishers now require a formal declaration of \gls{AI} use. \emph{Please check your school's specific regulations regarding Generative \gls{AI}.}
\end{itemize}


\section{Configuration}

The rendering of the statement is controlled by the \latexinline{\ntsetup{print/aidisclosure=OPTION}} option in the file \verb!0-Config/1_novathesis.tex!.

Possible values for \latexinline{OPTION} are:

\medskip
\noindent
\begin{tabularx}{\textwidth}{>{\bfseries\ttfamily}lX}
  \toprule
    false & Omit the AI Disclosure Statement entirely.\\
    aidisclose & (Default) Render the \emph{standard} checklist using the \href{https://github.com/joaomlourenco/aidisclose}{\texttt{aidisclose} package}.\\
    filename.tex & Render a custom file (e.g., \texttt{aidisclosure.tex}) from the \texttt{1-FrontMatter/} folder.\\
  \bottomrule
\end{tabularx}


\section{The Standard aidisclose Checklist}

The \novathesis{} template uses the \href{https://github.com/joaomlourenco/aidisclose}{\texttt{aidisclose} package}~\cite{Lourenco:2025:aidisclose}, which follows the \emph{GAIDeT (Generative \gls{AI} Delegation Taxonomy)}~\cite{Suchikova:2025:GAIDeT}. This provides a structured checklist of tasks (e.g., brainstorming, proofreading, code generation) where \gls{AI} might have been employed.

\subsection{Using the Companion Website (Recommended)}

The most efficient way to generate your disclosure is to use the companion web site at \url{aidisclose.org} (see \Autoref{fig:aidisclose-org}).

\begin{enumerate}
  \item Visit \href{https://aidisclose.org}{aidisclose.org}.
  \item Fill in your name and the \gls{AI} tools used (e.g., ChatGPT, Claude, Gemini).
  \item Select the appropriate checkboxes in the \emph{Taxonomy of Delegated Tasks}.
  \item Click the green button to generate the \LaTeX\ snippet.
  \item Copy the snippet and replace the contents of \verb!0-Config/7-aidisclose.tex! in your project.
\end{enumerate}

\subsection{Manual Configuration}

Alternatively, you can manually edit \verb!0-Config/7-aidisclose.tex! by activating specific tags:
\begin{latexcode}
\AIDactivate{w:poly}  % For Polishing and Editing
\AIDactivate{s:gen}   % For Code Generation
\end{latexcode}


\section{What if No \glsfmtshort{AI} was Used?}

If you did \textbf{not} use any Generative \gls{AI} tools, you have two choices:
\begin{enumerate}
  \item \textbf{Deactivate it:} Set \latexinline{\ntsetup{print/aidisclosure=false}} in \verb!0-Config/1_novathesis.tex!.
  \item \textbf{Negative Declaration:} Some schools prefer an explicit statement that no \gls{AI} was used. In this case, create a custom file in \verb!1-FrontMatter/aidisclosure.tex! with your declaration and set \latexinline{\ntsetup{print/aidisclosure=aidisclosure.tex}}.
\end{enumerate}


\section{Placement in the Document}

By default, the \gls{AI} Disclosure Statement is printed in the \textbf{Front Matter}, typically appearing after the Integrity Statement and before the Abstracts. It is automatically included in the PDF when \texttt{docstatus} is set to \texttt{provisional} or \texttt{final}.
